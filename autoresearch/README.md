# DeepEarth autoresearch

A self-contained environment for autonomously researching and improving **DeepEarth**.

1. Clone `github.com/legel/deepearth`
2. `cd deepearth/autoresearch`
3. Read `autoresearch.md` + `science.md`.
4. `python -m deepearth.autoresearch.prepare`
5. Run `train.py`, score, edit, repeat.
